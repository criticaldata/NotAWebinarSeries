# TINAWS — This Is Not a Webinar

A website for **MIT Critical Data's "This Is Not a Webinar"** project — an initiative for people who are done pretending learning is a ladder.

> "We don't want to boil the ocean. We want to fry it."

This Is Not a Webinar is a recurring conversation series — part salon, part confessional, part coup-in-planning — for students, researchers, and curious humans who suspect that the way we learn, publish, and perform has gone badly wrong. It grew out of an impromptu Zoom call on May 24, 2025, convened by Leo Anthony Celi (MIT Critical Data). The intended output is rewired participants, not polished publications.
